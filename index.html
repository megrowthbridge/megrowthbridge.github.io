Make.com Integration Setup Guide

1. Import Blueprint

Import the pre-configured make-blueprint.json file into a new Make.com scenario. This sets up HubSpot, Google Calendar, Twilio, and Gmail modules instantly.

2. Copy Webhook URL

Create the scenario webhook, copy the URL, and paste it into Vapi's webhook configuration page under the "End of Call Webhook" option. Also configure it in your local .env file.

3. Run and Authorize

Authorize each system (HubSpot, Google Calendar, Twilio, Gmail) in Make.com by clicking "Add Connection" for each node. The data mappings are already configured to match Vapi's outputs.
